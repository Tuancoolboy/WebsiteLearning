# WebsiteLearning — HCMUT Study Hub

A static educational website built for students at **Ho Chi Minh City University of Technology (HCMUT / Bach Khoa)**. The site provides a centralized hub for accessing study materials, course documents, live streaming, and online/offline surveys.

🌐 **Live Demo:** https://tuancoolboy.github.io/WebsiteLearning/

---

## 📋 Overview

WebsiteLearning is a front-end only website that aggregates learning resources for HCMUT students. It covers key first-year subjects and provides quick access to lecture materials, instructor information, and group video content. The site also integrates Google Meet for live class streaming.

---

## ✨ Features

- **Responsive Navigation** — Mobile-friendly header with hamburger menu
- **Hero Banner** — Welcome section tailored for HCMUT students
- **Subject Categories** — Document hubs for:
  - Giải Tích (Calculus)
  - Vật Lý (Physics)
  - Máy Tính (Computer Science)
  - Hoá Học (Chemistry)
- **Instructor Profiles** — Showcases lecturers for each subject
- **Live Streaming** — Direct link to Google Meet for online classes
- **Blog / Resources Section** — Links to online survey, group video, and offline survey
- **Platform Stats** — 29.3K students enrolled, 32.4K classes completed, 100% satisfaction rate, 354+ instructors
- **Back to Top Button** — Smooth scroll navigation

## 🚀 Getting Started

# 2. Open in browser

open index.html
